# 🍊 CalFuel Mini App

Telegram Mini Application для отслеживания калорий.

## ✨ Возможности Mini App

- Просмотр нормы калорий на сегодня
- Красивый прогресс-бар с остатком
- Добавление приёмов пищи в один клик
- Своя база блюд (быстрое добавление)
- История за последние 7 дней
- Полный профиль с перерасчётом нормы
- Редактирование и удаление записей

## 🛠 Технологии

- **HTML + Tailwind CSS**
- **Vanilla JavaScript**
- Полная интеграция с Telegram WebApp (`tg.initDataUnsafe`)
- Адаптив под мобильные устройства
